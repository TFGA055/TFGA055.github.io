# Fix & Frame — Landing Page

Μοντέρνο single-page React site για κουφώματα, ντουλάπες, ηλεκτρολογικά και υδραυλικά.

## Τεχνολογίες

- React 18 με hooks
- Vite
- Tailwind CSS
- Lucide React για εικονίδια

## Πώς να το τρέξετε

1. Εγκαταστήστε Node.js (αν δεν υπάρχει ήδη).
2. Στον φάκελο του project (`Новая папка`) τρέξτε:

```bash
npm install
npm run dev
```

3. Ανοίξτε τον browser στη διεύθυνση που θα σας δώσει η Vite (π.χ. `http://localhost:5173`).

Το βασικό component είναι το `src/RenovationLandingPage.tsx` και γίνεται render από το `src/main.tsx`.
